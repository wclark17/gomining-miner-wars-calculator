Assumptions

Cycle boundaries are Tuesday 00:00 UTC to Tuesday 00:00 UTC. Solo BTC uses network-derived sats/TH/day, so it excludes transaction fees, pool luck, and GoMining-specific accounting adjustments.

Autofill uses public GoMining Miner Wars leaderboard data when the browser or hosting proxy allows the request. It can fill league block sizes and weighted average W/TH automatically; clan and player block wins need matching search text.

Autofilled fields are marked until you manually edit them, so it is obvious when live or cached data replaced your entered value.

In block-estimate mode, BTC reward is clan blocks won times BTC block size times your clan TH share. If My TH and Clan total TH are entered, that ratio is used; otherwise the percentage field is used. Personal GMT rewards are personal blocks won times GMT block size. Current-to-date mode extrapolates block wins to a full cycle based on elapsed time. Block-estimated rewards are treated as before costs by default.

In estimate mode, Miner Wars electricity and service are charged only against reward-equivalent TH from blocks won. The solo-equivalent reward portion uses your listed W/TH, and reward above estimated Mining Mode uses the league weighted average W/TH. The solo baseline still uses full-week mining-mode costs for the selected cycle power. In actual mode, the Miner Wars electricity and service inputs are used directly and scaled only when current-to-date mode is active.

Estimated solo net is floored at zero. Ratio outputs are hidden when solo net is zero because a ratio against zero is nonsense wearing a tiny hat.

Disclaimer: This calculator is an estimate for planning and comparison only. Miner Wars rewards, electricity, service, token prices, difficulty, and leaderboard data can change; verify final values in GoMining before making decisions. This is not financial advice.
