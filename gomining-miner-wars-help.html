Miner Wars Calculator Help

Definitions and assumptions for the weekly Miner Wars vs solo mining estimate.

Back to calculator

Cycle Timing

Miner Wars cycles are modeled as Tuesday 00:00 UTC to Tuesday 00:00 UTC. In current-to-date mode, the calculator scales block wins, entered rewards, and actual costs by elapsed cycle time. In full-week mode, values are used as entered.

Reward Inputs

Miner Wars league

Selects the public leaderboard to use for live autofill. Odyssey, Eclipse, and Horizon map to league IDs 1, 2, and 3. Dune I through Dune XXVI map to league IDs 5 through 30.

Autofill league data

Fetches current public leaderboard totals for the selected league. It can fill BTC block size, GMT block size, and league average W/TH. Add clan or player search text to fill block wins and TH for a specific row.

Clan search

Optional text matched against clan names. When a clan is found, autofill sets clan blocks won and clan total TH.

Player alias search

Optional text matched against player aliases. When a player is found, autofill sets cycle power TH, personal blocks won, and My TH in clan.

Clan blocks won

The number of BTC blocks your clan has won in the cycle so far, or for the full cycle when using full-week mode.

BTC block size

The BTC amount assigned to each clan block.

My % of clan TH

Your share of total clan power. The calculator uses this when the optional TH ratio fields are blank.

My TH / Clan TH

If both values are entered, the calculator uses My TH / Clan total TH instead of the percentage field.

Actual MW reward BTC

Use this for end-of-cycle comparisons when you have the actual BTC reward value.

Personal GMT rewards

Personal GMT reward is estimated as personal blocks won * GMT block size. Add a GMT price if you want that reward included in dollar comparisons.

Cost Inputs

Electricity $/kWh

The electricity rate used for estimates. GoMining examples commonly use 0.05 USD/kWh, while documentation describes a billing-period average range.

Maintenance discount %

The discount applied to estimated electricity and service costs.

Service $ / TH / day

The service charge used for weekly estimates. The calculator multiplies this by TH, days, and the discount factor.

Actual electricity and service

Use actual values after the cycle is complete or when you have them from the app.

Spell allowance GMT

Enter planned spell spending in GMT. The calculator converts it to USD with the GMT price field before subtracting it from the Miner Wars result.

Comparison Logic

The solo baseline estimates normal mining reward from whole network sats per TH per day, BTC price, cycle power, electricity, and service. The Miner Wars side uses either block-estimated reward or actual BTC reward as before-cost reward value, adds optional personal GMT value, subtracts Miner Wars electricity and service, then subtracts spell allowance converted from GMT to USD.

For Miner Wars maintenance estimates, electricity and service are charged only against reward-equivalent TH from blocks won. The solo-equivalent reward portion uses your listed W/TH. Reward above the estimated solo-mining equivalent uses the league average W/TH input. The solo baseline remains a full-week mining-mode estimate for the selected cycle power.
